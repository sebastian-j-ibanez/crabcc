# crabcc 🦀

The crabby C compiler.

### What is this?

A C compiler built while following [Writing a C Compiler](https://norasandler.com/book/) by Nora Sandler.

This is an educational compiler, not meant for serious use.

### Why?

1. Learn how to write a compiler.
2. Learn more about C.
3. Continue my mission to flood the internet with useless software.
